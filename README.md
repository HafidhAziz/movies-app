# the-movies
Simple Movies App for Home Test Purpose
* [Movies API](https://developer.themoviedb.org/) - Public API.

# What's inside
* Kotlin Programming Language
* Hilt Dependency Injection
* Room Database
* Paging 3
* Kotlin Coroutines
* MVVM Pattern
* Use Dependency Management
* Sample UI Test
* Sample Local Database Test
* Code with love
